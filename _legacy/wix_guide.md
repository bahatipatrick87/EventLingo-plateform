# EventLingo: Wix Guide & Presentation Tips

Since you are presenting on another laptop, you have two great options.

## Option A: Present the HTML File (Recommended)
This is the easiest way to guarantee it looks **exactly** like the impressive mockup I created.

1.  **Download the File:** Save the `index.html` file to a USB drive or email it to yourself/your teammate.
2.  **Open on Presentation Laptop:** Just double-click `index.html`. It will open in the browser (Chrome, Safari, Edge) and work perfectly offline.
3.  **Why this adds value:** It shows you went "above and beyond" by prototyping with code, which Computer Science professors usually love.

## Option B: Recreate in Wix
If you *must* use the Wix editor, follow these steps to make it look like the mockup.

### 1. Global Design
*   **Fonts:** Go to **Site Design > Text Styles**. Change headings to a modern sans-serif like **Poppins** or **Montserrat**.
*   **Colors:** Set your site palette to:
    *   Primary: Indigo (`#4F46E5`)
    *   Secondary: Violet (`#8B5CF6`)
    *   Background: White

### 2. The Hero Section (Home)
*   **Add Strip:** Click `+` > `Strip` > `Classic`.
*   **Background:** Change Strip Background > Color > Select a light grey/indigo gradient.
*   **Text:** Add a Huge Heading ("Break Language Barriers...").
*   **Buttons:** Add two buttons side-by-side.
    *   Button 1: Solid color (Indigo), rounded corners (30px radius).
    *   Button 2: Target styling (White background, Indigo border).

### 3. Contact Us Section
*   **Add Strip:** `+` > `Strip`. Make it White.
*   **Columns:** Click "Layout" > "2 Columns".
*   **Left Column (Text):** Add your title ("Get in touch") and contact info (Email, Location icons).
*   **Right Column (Form):**
    *   Click `+` > `Contact & Forms`.
    *   Select a clean, simple form.
    *   **Design:** Click the form > `Design` icon > `Customize Design`.
    *   **Shadow:** Add a "Drop Shadow" to the form container to make it "pop" off the page.
    *   **Corners:** Set corner radius to 15px.

### 4. Navigation
*   **Anchor Menu:** Since it's a single page design:
    *   Add **Anchors** to each strip (`+` > `Menu & Anchor` > `Anchor`). Name them "Home", "About", "Services", "Contact".
    *   Link your Site Menu items to these anchors.
